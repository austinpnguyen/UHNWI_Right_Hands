## How Information Moves (The Orchestration Funnel)
1. **The Mandate:** Founder -> CoS -> CEO.
2. **The Output:** CEO -> C-Suite -> `templates/master_plan_template.md`.
3. **The Crucible:** `master_plan_v1.md` is placed in `crucible_testing/`.
4. **The Feedback:** External AI Agents write rejections/approvals to `feedback_logs/`.
5. **The Audit:** The Shield reads `feedback_logs/` to verify truth.
6. **The Loop:** Audit -> CEO -> V2 -> Crucible -> Audit.
7. **The Execution:** Audit (Pass) -> CoS -> CIO (Funds) -> Founder.
