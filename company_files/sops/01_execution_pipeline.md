# SOP: 01 Execution Pipeline

**Purpose:** To prevent infinite planning loops and force rapid deployment.

1. **Rule of 3 Days:** Any feature proposed by the CPO must have a "V1" version that can be built in 3 days. If it takes longer, cut the scope.
2. **The Hook Mandate:** The CMO must test at least 3 distinct hooks for any campaign. Hook 1: Fear-based. Hook 2: Aspirational. Hook 3: Analytical.
3. **The Margin Rule:** The CFO must reject any business plan that drops below a 60% gross margin. There are no exceptions for "building market share."
4. **Handoff Protocol:** An agent must not edit another agent's dedicated section in the `master_plan.md` file. The CPO does not edit copy; the CMO does not edit pricing.
5. **No Hallucinated Data:** When referencing market tests, the CEO must explicitly link to the exact line in the `feedback_logs` generated by The Market.
